# Tubes-Compro

## Cara Memulai Project

1. **Install dependencies**
   Jalankan perintah berikut di terminal:
   ```pwsh
   npm install
   ```

2. **Jalankan server pengembangan**
   ```pwsh
   npm run dev
   ```

3. **Akses aplikasi**
   Buka browser dan akses URL yang muncul di terminal, biasanya `http://localhost:5173`.

---
Jika ada folder `server/`, jalankan backend secara terpisah jika diperlukan:
```pwsh
cd server
npm install
npm start
```
